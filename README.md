JGLFW
=====
JGLFW is a crossplatform (Windows, Linux, Mac OS X) wrapper for
 * GLFW, for window managment, OpenGL context creation and input handling
 * OpenGL, for graphics
 * OpenAL, for audio

The GLFW bindings where created manually using gdx-jnigen
